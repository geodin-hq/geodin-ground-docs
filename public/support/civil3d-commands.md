
# Commands
| Ribbon/Toolbar Icon | Command | Command Description |
|---------------------|---------|---------------------|
| ![GeoDin® Ground database select file](icons/importer_icon-16.png)  | GEODIN_DATABASE_SELECT_FILE | Select a GeoDin® database file from your hard drive and include it in the user registry for use within Civil 3D|
| ![GeoDin® Ground location selector](./icons/Location_selector-16.png) |GEODIN_SELECT_LOCATION |Select locations from a specific GeoDin® project to display in Civil 3D |
| ![GeoDin® Ground draw boreholes](./icons/draw_boreholes-16.png)|GEODIN_DRAW_BOREHOLES | Creates detailed boreholes with lithological descriptions and annotations|
|  |GEODIN_DRAW_BOREHOLES_BASIC |Creates basic boreholes without lithological descriptions or annotations |
|  |GEODIN_DRAW_BOREHOLES_OPT |Creates detailed boreholes with options to use lithological descriptions or ground units for section division |
|  |GEODIN_DRAW_SURFACES |Generate surfaces and solids based on the lithological layers of all boreholes |
|  ![GeoDin® Ground draw surfaces and solids](./icons/Surfaces-16.png)|GEODIN_DRAW_SURFACES_FROM_SELECTION |Generate surfaces and solids based on the lithological layers of selected boreholes |
|  |GEODIN_OPEN_ALL_DOCUMENTS |Open GeoDin® Document Viewer with all available documents in the database |
|  ![GeoDin® Ground document viewer](./icons/Documents-16.png)|GEODIN_OPEN_ALL_DOCUMENTS_OFFLINE | Open GeoDin® Document Viewer with all exported documents stored in the Autodesk Constructive Cloud|
|  |GEODIN_OPEN_BOREHOLE_DOCUMENTS |Open GeoDin® Document Viewer with documents of selected boreholes|
|  |GEODIN_OPEN_BOREHOLE_DOCUMENTS_OFFLINE |Open GeoDin® Document Viewer with documents of selected boreholes stored in the Autodesk Constructive Cloud |
